## Data Structures

### Lists

- [1, 2, 3]
- Reading `O(1)`
- Searching, Inserting, Deleting `O(n)`
- `int` keys

### Sets

- {1, 2, 3}
- `unordered` and `no duplicates`
- Inserting `O(1)`

- Searching, Deleting `O(n)`
