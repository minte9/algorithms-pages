## Data Structures

### Lists

- [1, 2, 3]
- Reading `O(1)`
- Searching, Inserting, Deleting `O(n)`
- `int` keys
- items.`append()`

### Sets

- {1, 2, 3}
- `unordered` and `no duplicates`
- Inserting `O(1)`
- Reading , Inserting `O(1)`
- Searching, Deleting `O(n)`
- items.`add()`

### Hash Tables

- hashing encoding for `indexes`
- time complexity `O(1)`
- python `dictionary`
